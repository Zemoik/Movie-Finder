# 🎬 Movie Finder

A cross-platform mobile app that lets users discover popular movies by genre with real-time data from the TMDb API.

## Features
- Browse movies by genre with live TMDb API data
- Dynamic movie cards showing posters, ratings, and descriptions
- Watchlist interaction and multi-screen navigation
- Optimized API calls for performance

## Tech Stack
React Native · JavaScript · TMDb REST API · React Navigation

## Setup
```bash
npm install
npx expo start
```
Get a free TMDb API key at [themoviedb.org](https://www.themoviedb.org/settings/api) and add it to a `.env` file.
